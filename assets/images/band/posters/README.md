# Show Posters Directory

This directory contains poster images for Lake Superior band shows.

## File Naming Convention

Use the following format: `YYYY-MM-DD-show-name.jpg`

Examples:
- `2026-02-27-fire-flow-friday.jpg`
- `2025-08-08-better2gether.jpg`
- `2025-11-01-harvest-kickoff.jpg`

## Image Format

- **Format**: JPG or PNG
- **Orientation**: Portrait (story/poster style)
- **Recommended size**: 1080px wide or similar Instagram story dimensions
- **Quality**: High enough for fullscreen modal viewing

## Adding New Shows

When adding a new show to band.html:
1. Add the poster image to this directory using the naming convention
2. Reference it in the HTML with both `src` and `data-full-res` attributes
3. Update this README with the new poster filename
