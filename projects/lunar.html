Lunar Lander in Unity Engine

A modern take on the classic Lunar Lander game, built with Unity. This project combines procedural generation with physics-based gameplay and cinematic effects.

Key Features

Technical Highlights

  • Perlin Noise: Procedurally generated landscape for unique terrain each game
  • Cinematography: Dynamic camera zoom that adjusts based on player movement and landing zones
  • Particle Effects: Realistic plasma-like rocket torch with dynamic lighting and physics interactions
