Challenges Faced

Staticman Integration

Implementing comments on a static site without a backend

Set up Staticman to handle form submissions and automatically generate pull requests to add comments as data files

Jekyll Template Customization

Modifying an existing template without breaking its core functionality

Took an incremental approach to modifications, creating component overrides rather than directly modifying template files

Build Configuration

Configuring the correct build settings for Netlify deployment

Created a detailed Netlify configuration file with precise build commands and environment settings
