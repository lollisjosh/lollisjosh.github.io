Project Overview

This was my first professional portfolio website, built as a personally driven project during summer 2023 specifically to showcase my skills while applying for internships. The site was built using a Jekyll template that I customized to fit my specific needs and aesthetic preferences.

  • Markdown

Hosting & Integration

  • Form Handling

Implementation Details

Site Architecture

  • Template-Based Page Generation
  • Custom Layouts for Different Content Types

  • Content/Code Separation

Customizations

  • Modified Layout Structure
  • Custom Color Scheme & Typography
  • Project Gallery System
  • Responsive Navigation

Challenges Faced

Staticman Integration

Implementing comments on a static site without a backend

Set up Staticman to handle form submissions and automatically generate pull requests to add comments as data files

Jekyll Template Customization

Modifying an existing template without breaking its core functionality

Took an incremental approach to modifications, creating component overrides rather than directly modifying template files

Build Configuration

Configuring the correct build settings for Netlify deployment

Created a detailed Netlify configuration file with precise build commands and environment settings
