Project Overview

Project GAL is a first-person shooter game focused on target practice and aim training. Built with Unity, it features dynamic target systems, custom animations, and visual effects to create an engaging training environment.

Technologies Used

Core Technologies

  • Unity Engine
  • C#
  • Unity Physics
  • Unity Particle System

Development Tools

  • Git/GitHub
  • Unity Asset Store
  • Unity ProBuilder
